# Security Policy

## Supported Versions

Only the latest release receives security fixes.

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead, email the maintainer directly or use GitHub's private vulnerability reporting:
[https://github.com/akib/docker-tui/security/advisories/new](https://github.com/akib/docker-tui/security/advisories/new)

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix if any

You can expect a response within 72 hours.
